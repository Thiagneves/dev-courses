# Git - Estudos e Prática

Este repositório reúne anotações e exemplos práticos para o aprendizado de Git, focando no controle de versão e no gerenciamento de projetos.

## 📚 Conteúdo

- Inicialização de repositórios
- Controle de versões
- Commits e histórico
- Branches e merge
- Boas práticas com Git

## 🎯 Objetivo

Compreender e aplicar os conceitos fundamentais de versionamento para organizar e gerenciar projetos de forma eficiente.
