# Curso Git - Fundamentos

Este repositório contém anotações e exemplos práticos do curso básico de Git.

## 📚 Conteúdo

- Inicialização de repositório (`git init`)
- Adição de arquivos (`git add`)
- Criação de commits (`git commit`)
- Verificação de status (`git status`)
- Histórico de commits (`git log`)
- Uso básico de branches

## 🎯 Objetivo

Compreender os conceitos fundamentais do Git e aprender a controlar versões de forma simples e eficiente.
