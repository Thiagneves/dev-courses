# Lógica de Programação - IFRS

Este repositório contém exercícios e exemplos desenvolvidos durante o curso de Lógica de Programação do IFRS, utilizando Portugol.

## 📚 Conteúdo

- Algoritmos em Portugol
- Estruturas de decisão (if/else)
- Estruturas de repetição (for, while)
- Vetores e matrizes
- Variáveis e tipos de dados
- Resolução de problemas práticos

## 🎯 Objetivo

Desenvolver o raciocínio lógico e compreender os fundamentos da programação por meio da construção de algoritmos em Portugol.
