# Lógica de Programação

Este repositório reúne exercícios e exemplos voltados ao estudo de lógica de programação, utilizando diferentes abordagens para o desenvolvimento do raciocínio lógico.

## 📚 Conteúdo

- Algoritmos básicos
- Estruturas de decisão (if/else)
- Estruturas de repetição (for, while)
- Variáveis e tipos de dados
- Vetores e matrizes
- Resolução de problemas

## 🎯 Objetivo

Fortalecer a base lógica necessária para a programação, facilitando o aprendizado de diferentes linguagens e tecnologias.
