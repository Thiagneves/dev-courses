# Programação e Algoritmos em C - IFNMG

Este repositório contém exercícios e exemplos desenvolvidos em linguagem C durante a disciplina de Programação e Algoritmos do IFNMG, além de um projeto final aplicado.

## 📚 Conteúdo

- Algoritmos em C
- Estruturas de decisão (if/else)
- Estruturas de repetição (for, while)
- Variáveis e tipos de dados
- Manipulação básica de entrada e saída
- Resolução de problemas
- Projeto final

## 🎯 Objetivo

Desenvolver o raciocínio lógico e aplicar os fundamentos da programação utilizando a linguagem C.
