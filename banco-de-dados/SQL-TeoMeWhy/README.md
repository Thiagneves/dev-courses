# 🗄️ Banco de Dados - Curso Teo My Why

Este repositório organiza meus estudos em **Banco de Dados** no curso **Teo My Why**, incluindo exercícios, exemplos e projetos práticos. O objetivo é reforçar conceitos fundamentais e desenvolver habilidades de modelagem, consulta e manipulação de dados.

---

## 📚 Conteúdo

- Introdução a Banco de Dados 
- Modelagem de dados (Entidades, Relacionamentos, Diagramas ER) 
- SQL básico e avançado 
  - `SELECT`, `INSERT`, `UPDATE`, `DELETE` 
  - `JOIN`, `GROUP BY`, `ORDER BY` 
  - Subqueries e Views 
- Normalização de dados 
- Exercícios e projetos práticos do curso 

---

## 🎯 Objetivo

Centralizar todo o aprendizado de Banco de Dados do curso **Teo My Why**, facilitando a revisão e servindo como referência prática para consultas, projetos e aplicações reais.

---

## 🚀 Estrutura do Repositório

- `/Exercicios` – Exercícios realizados durante o curso 
- `/Projetos` – Projetos completos aplicando os conceitos de Banco de Dados 
- `/Diagramas` – Modelos ER e esquemas de banco de dados
